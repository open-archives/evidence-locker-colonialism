## Archive Standard

This repository strictly adheres to the Evidence Locker documentation standards:
1. **Primary Sourcing:** All assertions are tracked directly back to primary sources (treaties, declassified minutes, census records).
2. **Neutral Taxonomy:** Claims are mapped against evidence with explicit qualifiers and data gaps highlighted in tabular form.
3. **Decentralized Modularity:** Created as a dedicated sibling repository to prevent bloat while preserving cross-references.
